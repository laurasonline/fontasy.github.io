What is it?|

Fontasy is a font application created with designers in mind,
to help minimise the time taken when looking for good fonts.

How does it work?|

During the set up process for this
application it allows you to choose
which font styles you prefer or use,
this personalises the user experience.

What's in it for us?|

We wanted our users to experience
something more than just lists of
different fonts, so we implimented a
"Font of the day" feature targeted
towards the users taste.

Pricing?|

As this application features various fonts varying from free to premium,
this app does cost for certain features.
